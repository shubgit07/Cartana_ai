1. Project Overview
Objective:
Build a multi-modal AI-powered platform that converts managerial voice notes or text input 
into structured, assignable tasks with extracted deadlines, priorities, and assignees.
2. Problem Statement
Managers spend significant time converting informal discussions into structured tasks.
This platform reduces inefficiency and improves clarity by automating task extraction.
3. Core Features (MVP)- Record Live Audio- Upload Audio File- Direct Text Input- AI Task Extraction- Deadline Normalization- Kanban Board (To Do / In Progress / Done)
4. Tech Stack
Frontend: React
Backend: FastAPI
Database: PostgreSQL
AI: Whisper (Speech-to-Text), LLM API, dateparser
